# Looker MCP

This project demonstrates how to set up and configure Looker for use with the Model Context Protocol (MCP) and Gemini CLI. It allows for enhanced and extended Looker operations over MCP, leveraging the MCP Toolbox to connect Looker's data platform capabilities with the interactive and AI-driven features of Gemini CLI.

This project contains a collection of Gemini CLI skills for interacting with Google Cloud services, with a special focus on Looker integration.
